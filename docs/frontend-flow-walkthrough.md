## 走查说明

- **可达**：该角色可进入页面并完成主操作  
- **阻断**：路由守卫或业务规则阻止操作  
- **部分**：页面可达但流程不完整或缺引导  
- **只读**：可查看不可改

## P0 改进验收（本迭代）

| 项 | 验收标准 |
|----|----------|
| 黑名单起草拦截 | 选黑名单相对方 Toast 告警；提交弹出 MessageBox 并阻止 |
| OCR 人工确认 | `party_parse_warning` 或 confidence&lt;0.6 时须勾选确认才可提交 |
| 条款比对侧栏 | 侧栏可见「条款比对」；无合同时进入选合同引导页 |
| 审批 AI 风险 | 待办列展示风险等级；high 通过前二次确认 |
| 生命周期条 | 合同详情展示 draft→…→archived 进度与「下一步」按钮 |
