## 一、浪潮品牌体系分析

### 1.1 品牌资产识别

| 资产类型 | 内容 | 用途 |
|---------|------|------|
| **品牌Logo** | Inspur 浪潮标志 | 产品Logo、Favicon |

## 二、前端品牌集成方案

### 2.1 颜色系统

#### 当前颜色
```css
:root {
  --primary: #1d4ed8;     /* 原蓝色 */
}
```

#### 浪潮品牌色
```css
:root {
  /* 浪潮品牌主色 */
  --inspur-primary: #0066CC;
  --inspur-primary-light: #3399FF;
  --inspur-primary-dark: #004C99;
  
  /* 海岳系列辅助色 */
  --inspur-success: #00C853;
  --inspur-warning: #FFAB00;
  --inspur-danger: #D50000;
  
  /* 中性色（企业级） */
  --inspur-gray-50: #F5F7FA;
  --inspur-gray-200: #E4E7ED;
  --inspur-gray-500: #909399;
  --inspur-gray-800: #303133;
  --inspur-gray-900: #1F2D3D;
}
```

### 2.2 Logo 集成

#### 文件准备
- `frontend/public/logo.svg` - 主Logo（彩色版）
- `frontend/public/logo-white.svg` - 浅色Logo（用于深色背景）
- `frontend/public/favicon.ico` - 网站图标
- `frontend/public/favicon-32x32.png` - 备用图标

#### 侧边栏更新
当前代码 (`AppLayout.vue`):
```vue
<div class="sidebar-brand">
  <div class="brand-icon">合</div>
  <div>
    <div class="brand-title">合同审批管理平台</div>
    <div class="brand-sub">Contract Lifecycle</div>
  </div>
</div>
```

更新后:
```vue
<div class="sidebar-brand">
  <img src="/logo-white.svg" alt="浪潮" class="brand-logo" />
  <div>
    <div class="brand-title">浪潮合同管理</div>
    <div class="brand-sub">Inspur inContract</div>
  </div>
</div>
```

### 2.3 页面标题更新

**当前**: `合同审批管理平台`
**更新为**: `浪潮合同审批管理平台 | Inspur inContract`

### 2.4 Element Plus 主题覆盖

```css
/* element-theme.css */
:root {
  --el-color-primary: #0066CC;
  --el-color-primary-light-3: #3399FF;
  --el-color-primary-dark-2: #004C99;
  --el-color-white: #ffffff;

  --el-bg-color: #F5F7FA;
  --el-text-color-primary: #303133;
}

/* 实心主按钮渐变；勿写 .el-button--primary 全局 background，否则会破坏 plain 变体 */
.el-button--primary:not(.is-plain):not(.is-link):not(.is-text):not(.is-dashed) {
  background: linear-gradient(135deg, #0066CC 0%, #004C99 100%);
  border-color: #0066CC;
  color: #fff;
}

/* 菜单样式 */
.el-menu {
  background-color: #1F2D3D;
}

.el-menu-item.is-active {
  background-color: #0066CC;
}
```

4. **国际化**：预留中英文切换接口
