## 一、CUAD 标准化风险分类体系

### 借鉴来源
Atticus CUAD (Contract Understanding Atticus Dataset) 是合同理解的标准基准数据集，包含 40+ 种合同风险类型的检测任务。

### 原型实现
| 实现内容 | 位置 | 说明 |
|---------|------|------|
| CUAD 风险类型标签 | 评审工作台 4 个审查面板 | 每个面板显示 CUAD 编号 + 风险类型（如 CUAD-07 付款条件） |
| 风险类型筛选器 | AI审查报告页 | 下拉框支持按风险类型筛选（付款条件/违约金/赔偿上限/管辖法院等） |
| CUAD 颜色标签 | 全局 CSS | 7 种颜色对应 7 种风险类型（payment/penalty/indemnity/jurisdiction/confidentiality/termination/warranty） |

### 效果
```
评审工作台审查面板：
┌─────────────────────────────────────┐
│ 💳 付款条件 | 💰 预付款比例         │
│ CUAD-07 | 置信度: 高                │
│                                       │
│ ⚠️ 预付款比例偏高 (40%)              │
│ 📖 内部财务制度：预付款比例原则上...  │
│ ✓ 确认风险 | ✏️ 修正 | ⊘ 忽略       │
└─────────────────────────────────────┘
```

## 二、律师级修订标注格式

### 借鉴来源
律师标准合同修订格式：原文 → 问题诊断 → 法律依据 → 修改建议 → 谈判话术

### 原型实现
风险详情弹窗升级为 6 步律师级标注：

| 步骤 | 字段 | 说明 |
|------|------|------|
| 1 | 原文定位 | 精确到条款原文 |
| 2 | 问题诊断 | CUAD 风险类型 + 风险等级 + 置信度 + 触发条件 |
| 3 | 法律依据 | 具体法条 + 司法实践标准 |
| 4 | 修改建议 | 原文 vs 建议修改（前后对比） |
| 5 | 谈判话术 | 与对方沟通的标准话术 |
| 6 | 市场基准 | 行业基准 + 偏离度分析 |

### 效果
```
律师级风险详情弹窗：
┌─────────────────────────────────────────────┐
│ 📄 第一步：原文定位                          │
│ 甲方在合同签订后 5 个工作日内支付...40%...   │
│                                             │
│ 🔍 第二步：问题诊断                          │
│ 预付款比例 40% 显著高于公司标准（≤30%）...   │
│ CUAD-07 付款条件风险 | 中风险 | 置信度: 高  │
│                                             │
│ ⚖️ 第三步：法律依据                          │
│ 《内部财务管理制度》第 12 条...              │
│                                             │
│ ✏️ 第四步：修改建议                          │
│ 原文：甲方支付合同总额的 40%...              │
│ 建议：甲方支付合同总额的 30%...              │
│                                             │
│ 💬 第五步：谈判话术                          │
│ "根据我司财务管理制度，标准预付款比例为...   │
│                                             │
│ 📊 第六步：市场基准对比                      │
│ 行业基准：预付款比例平均 25%                 │
│ 偏离度：40% vs 25% = +60% 偏离               │
└─────────────────────────────────────────────┘
```

## 三、市场基准对比

### 借鉴来源
与市场基准对比，显示条款偏离度。

### 原型实现
| 实现位置 | 内容 |
|---------|------|
| AI审查报告风险项 | 每条风险下方显示行业基准 + 当前值 + 偏离度 |
| 风险详情弹窗 | 第 6 步市场基准对比（行业基准/偏离度/历史合同） |
| 条款比对页面 | 标准模板 vs 上传合同差异高亮 |

### 效果
```
📊 行业基准：预付款比例平均 25%（范围 20%-30%）
📊 偏离度：40% vs 25% = +60% 偏离（红色高亮）
📊 历史合同：近 12 个月平均预付款 28%
```

## 四、合同类型自适应审查规则

### 借鉴来源
不同合同类型使用不同审查规则集。

### 原型实现
在 AI审查报告中增加"合同类型专属审查规则"区域：

```
🎯 采购合同专属审查规则
├─ ✅ 供应商资质审查：已验证（营业执照/资质证书）
├─ ✅ 交货期条款：已包含（逾期违约金）
├─ ✅ 验收标准：已包含（设备清单附件）
├─ ⚠️ 预付款比例：40%（采购合同标准 ≤30%）
├─ ⚠️ 质保条款：缺失（采购合同建议包含）
└─ ⚠️ 违约金比例：0.5%/日（采购合同标准 0.05%/日）
```

### 扩展性
- 支持 6 种合同类型：采购/销售/劳务/合作/保密/法务制式
- 每种类型有专属审查规则集
- 规则随合同类型自动切换
