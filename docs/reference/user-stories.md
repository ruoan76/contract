# 用户故事与验收标准

> 版本：V1.0 | 日期：2024-01-15

## 一、合同起草（US-001 ~ US-010）

### US-001：空白起草合同
**作为** 业务人员，**我希望** 手动创建空白合同，**以便** 起草非标准合同。

**验收标准：**
- [ ] 点击"新建合同"进入空白起草页
- [ ] 必填字段：合同名称、合同类型、相对方、合同金额
- [ ] 选填字段：开始日期、结束日期、备注
- [ ] 支持上传合同文件（PDF/Word/图片，最大 50MB）
- [ ] 点击"保存草稿"保存当前编辑内容
- [ ] 点击"提交审批"进入审批流程

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 正常创建 | 填写必填字段 + 提交 | 合同创建成功，进入审批流程 |
| 缺必填项 | 未填合同金额 + 提交 | 提示"合同金额为必填项" |
| 文件过大 | 上传 60MB 文件 | 提示"文件大小不能超过 50MB" |
| 草稿保存 | 编辑 50% + 保存草稿 | 草稿保存成功，可继续编辑 |

---

### US-002：模板起草合同
**作为** 业务人员，**我希望** 从模板库选择标准模板，**以便** 快速起草合同。

**验收标准：**
- [ ] 新建合同时显示模板选择页
- [ ] 模板按类型分类：采购、销售、劳务、保密、NDA
- [ ] 选择模板后自动填充条款文本
- [ ] 可变字段标记为 `{金额}`、`{日期}`、`{相对方}`，支持编辑
- [ ] 模板起草 3 步完成：选模板→填关键信息→提交
- [ ] 支持预览模板内容

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 选择采购模板 | 选"采购合同模板"→填金额/日期→提交 | 合同创建成功，条款自动填充 |
| 模板预览 | 点击"预览" | 显示模板完整内容，可变字段高亮 |
| 必填字段为空 | 未填金额→提交 | 提示"请填写可变字段：金额" |

---

### US-003：历史合同引用
**作为** 业务人员，**我希望** 引用已归档合同作为新合同基础，**以便** 避免重复起草。

**验收标准：**
- [ ] 新建合同页显示"引用历史合同"入口
- [ ] 支持按合同编号/名称搜索历史合同
- [ ] 引用后复制合同条款，金额/日期等字段可重新编辑
- [ ] 新合同与原合同建立关联关系

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 引用历史合同 | 搜索"CTR-2023-001"→引用→修改金额→提交 | 新合同创建成功，关联原合同 |
| 归档合同不可引用 | 引用状态为"已废止"的合同 | 提示"该合同已废止，不可引用" |

## 二、模板管理（US-011 ~ US-020）

### US-011：创建标准模板
**作为** 法务专员，**我希望** 上传标准合同并标记可变字段，**以便** 业务人员起草时引用。

**验收标准：**
- [ ] 支持上传 Word/PDF 文件作为模板
- [ ] 支持在线编辑模板文本
- [ ] 支持标记可变字段：`{金额}`、`{日期}`、`{相对方}`、`{交付周期}`
- [ ] 模板需指定类型：采购/销售/劳务/保密/NDA
- [ ] 模板需填写：名称、描述、适用场景

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 创建采购模板 | 上传文件→标记{金额}/{日期}→保存 | 模板创建成功，状态为"草稿" |
| 可变字段语法错误 | 标记为 `{金额`（缺右括号） | 提示"可变字段语法错误，请使用{字段名}格式" |

---

### US-012：模板发布审批
**作为** 法务负责人，**我希望** 审批新模板/模板修订，**以便** 确保模板质量。

**验收标准：**
- [ ] 新模板/模板修订需提交审批
- [ ] 审批人可查看模板变更对比
- [ ] 审批通过后模板状态变为"已发布"
- [ ] 审批拒绝后模板状态变为"已拒绝"，起草人可修改后重新提交

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 审批通过 | 查看变更对比→点击"通过" | 模板状态变为"已发布"，业务可引用 |
| 审批拒绝 | 点击"拒绝"→填写拒绝原因 | 模板状态变为"已拒绝"，起草人收到通知 |

---

### US-013：模板废止
**作为** 法务专员，**我希望** 废止旧版模板，**以便** 避免业务人员使用过时模板。

**验收标准：**
- [ ] 支持废止已发布模板
- [ ] 废止后模板不可被新建合同引用
- [ ] 已引用废止模板的合同不受影响
- [ ] 废止操作需填写原因

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 废止模板 | 选择模板→点击"废止"→填写原因 | 模板状态变为"已废止"，不可引用 |
| 引用废止模板 | 新建合同→选择已废止模板 | 提示"该模板已废止，请选择其他模板" |

### US-023：黑名单管理
**作为** 法务专员，**我希望** 管理相对方黑名单，**以便** 起草时自动拦截高风险相对方。

## 四、AI 审查（US-031 ~ US-040）

### US-031：发起 AI 审查
**作为** 起草人，**我希望** 提交合同后自动触发 AI 审查，**以便** 快速识别风险。

**验收标准：**
- [ ] 合同提交审批后自动触发 AI 审查
- [ ] 审查耗时 <30 秒（10 页以内合同）
- [ ] 审查完成后显示：风险等级、风险评分、审查报告
- [ ] 支持手动重新触发 AI 审查（合同修订后）

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 自动触发 | 提交合同→等待审查 | 30 秒内显示审查结果 |
| 手动触发 | 点击"重新审查" | 重新触发 AI 审查，覆盖旧结果 |
| 超时处理 | 审查超过 60 秒 | 提示"审查超时，请稍后查看" |

---

### US-032：查看审查报告
**作为** 审批人，**我希望** 查看 AI 审查报告，**以便** 快速判断合同风险。

**验收标准：**
- [ ] 审查报告显示：整体风险等级、风险评分、条款审查详情
- [ ] 高风险条款红色标记，中风险黄色标记，低风险绿色标记
- [ ] 每条审查结论附带：风险描述、法律依据、修改建议
- [ ] 支持一键导出审查报告（PDF/Word）

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 查看报告 | 打开合同详情→查看审查报告 | 显示结构化审查报告 |
| 导出报告 | 点击"导出报告"→选择格式 | 下载 PDF/Word 文件 |
| 高风险条款 | 查看高风险条款详情 | 红色标记，显示风险描述 + 法律依据 |

---

### US-033：法务复核 AI 审查
**作为** 法务专员，**我希望** 复核 AI 审查结论，**以便** 修正误报/漏报。

**验收标准：**
- [ ] 法务可确认/修正 AI 审查结论
- [ ] 支持标记"AI 误报"、"AI 漏报"
- [ ] 修正后更新审查报告，标记"法务已复核"
- [ ] 复核记录用于模型优化

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 确认 AI 结论 | 查看 AI 审查→点击"确认" | 审查报告标记"法务已复核" |
| 修正 AI 结论 | 修改风险等级→填写修正原因 | 审查报告更新，记录修正历史 |
| 标记误报 | 点击"AI 误报"→填写原因 | 记录误报，用于模型优化 |

## 五、审批流程（US-041 ~ US-050）

### US-041：提交审批
**作为** 起草人，**我希望** 提交合同进入审批流程，**以便** 尽快完成签署。

**验收标准：**
- [ ] 合同提交后自动匹配审批流程
- [ ] 显示审批流程节点 + 当前节点
- [ ] 提交后通知下一审批人

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 小额合同 | 提交 5 万采购合同 | 匹配简易流程：部门主管→法务→用印 |
| 大额合同 | 提交 200 万采购合同 | 匹配特殊流程：部门主管→法务→财务→高管→董事会→用印 |
| 提交通知 | 提交合同 | 下一审批人收到待办通知 |

---

### US-042：审批操作
**作为** 审批人，**我希望** 快速审批合同，**以便** 不阻塞业务进度。

**验收标准：**
- [ ] 支持操作：通过、拒绝、退回、委托、会签
- [ ] 通过/拒绝/退回需填写审批意见（可选）
- [ ] 委托需选择委托对象 + 委托期限
- [ ] 会签需所有会签人通过后才能进入下一节点

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 通过 | 点击"通过"→填写意见→提交 | 流转到下一节点 |
| 拒绝 | 点击"拒绝"→填写原因→提交 | 流程终止，起草人收到拒绝通知 |
| 退回 | 点击"退回"→填写原因→提交 | 合同返回起草人，可修改后重新提交 |
| 委托 | 点击"委托"→选择委托人→设置期限 | 委托对象获得审批权限 |

---

### US-043：待办列表
**作为** 审批人，**我希望** 查看待办审批列表，**以便** 快速处理待审批合同。

**验收标准：**
- [ ] 显示待审批合同列表，按紧急程度排序
- [ ] 每条显示：合同名称、金额、提交人、提交时间、紧急程度
- [ ] 支持快速审批（不进入详情页）
- [ ] 支持筛选：按合同类型、金额、紧急程度

**测试用例：**
| 用例 | 输入 | 预期结果 |
|------|------|---------|
| 查看待办 | 打开待办列表 | 显示待审批合同，按紧急程度排序 |
| 快速审批 | 勾选合同→点击"批量通过"→填写意见 | 多份合同一次性通过 |
| 筛选 | 选择"采购合同"→搜索 | 仅显示采购合同 |

---

### US-044：超时提醒与升级
**作为** 系统，**我希望** 自动提醒超时审批并升级，**以便** 避免流程卡住。

| 72h 自动通过 | 审批人 72h 未审批 | 自动通过，记录"超时自动通过" |
