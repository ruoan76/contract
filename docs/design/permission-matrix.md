## 4. 原型演示说明（重要）

**高保真原型与正式产品权限行为不同：**

| 项目 | 正式环境 | 原型 `prototype/` |
|------|----------|-------------------|
| 侧栏菜单 | 按 RBAC 隐藏无权限项 | **17 项全部展示** |
| 无权限菜单 | 不显示 | 显示为半透明 + 🔒 |
| 点击无权限菜单 | 403 / 不进入 | **仍可进入页面**（便于走查 UI） |
| 顶栏「新建合同」 | 按角色 | 始终显示（无权限时半透明） |
| 角色切换 | 切换菜单集 | 切换用户信息与 🔒 标记 + Toast 提示 |

实现位置：

- `js/01-data.js`：`ALL_NAV_PAGES`、`roleConfig[].visiblePages`
- `js/02-app.js`：`switchRole()` 仅加 `nav-restricted` 类，不 `display:none`
- `css/main.css`：`.nav-item.nav-restricted`

顶栏角色选择器用于**模拟身份**（姓名/部门），便于演示不同视角，**不代表原型隐藏菜单**。

正式开发按本矩阵 + API 鉴权 + 数据范围（部门/本人）实现，见 [api-spec.md](./api-spec.md)。
