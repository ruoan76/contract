# V1 演示脚本（15 分钟）

> 版本：1.2.0 | 日期：2026-05-18 | 原型：`prototype/index.html`（构建：`prototype/build.py`）  
> 导航与权限以 [DESIGN_STATUS.md](./DESIGN_STATUS.md) §3、[permission-matrix.md](./permission-matrix.md) §4 为准。

## 准备

- 浏览器打开原型；顶栏角色下拉切换身份（原型 **不隐藏**侧栏，无权限项带 🔒）
- 样例合同：DEMO-01 / DEMO-03 / DEMO-05 及默认数据

## DEMO-01 简易流程（约 3 分钟）

1. 侧栏 → **新建合同**，金额填 `80000`，类型采购
2. **提交审批** → 查看「流程匹配」弹窗（简易流程节点）
3. 角色切 **部门主管** → **待办审批** → 通过
4. 角色切 **法务** → **评审工作台**（仅法务 Tab）→ 通过
5. **用印管理** → 确认用印

## DEMO-02 标准评审（约 5 分钟）

1. 打开合同 **CTR-20240112-0007** 或 DEMO-05（32 万）
2. 侧栏 **审查报告** → 查看 65%/52% 置信度样例 → 标记误报/漏报
3. **评审工作台** → 法务 → 财务 → 高管 依次演示
4. **归档台账** 筛选

## DEMO-03 特殊流程 + 董事会（约 2 分钟）

1. **审批配置** 查看 >100 万阈值
2. **合同列表** → 合同 **DEMO-03**（250 万）→ 操作 **审批历史**（下钻页，无侧栏菜单）→ 时间线含董事会节点

## DEMO-04 黑名单（约 2 分钟）

1. **相对方管理** 查看黑名单
2. **新建合同** 选择黑名单相对方（若已配置）或演示警告文案

## DEMO-05 退回修订（约 3 分钟）

1. **评审工作台** → **退回修改**
2. **修订工作台** → 接受/拒绝建议 → 再提交
